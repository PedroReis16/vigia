# Older fall

Repositório destinado ao desenvolvimento ao sistema para monitormaneto de quedas de pessoas gerando um tipo de notificação. O nome older fall é meramente ilustrativo e temporário



## Criação do ambiente virtual

O ambiente virtual é uma estrutura que permite a execução completa do código, sem a necessidade de instalação completa dos códigos dentro do computador.

- **Criando o ambiente virtual**

Execute o seguinte comando no terminal: 

```
py -3.14 -m venv .venv
```

Ao executar o comando no terminal, irá ser criado uma pasta para ambiente virtual do projeto, nesse exemplo para a versão 3.14.

- **Ativando o ambiente virtual**

Após criar o ambiente virtual, para utilizar o ambiente, execute o seguinte comando no terminal

```
./.venv/Scripts/Activate.ps1
```
